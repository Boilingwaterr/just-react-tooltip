## interface
<h5>Tooltip render = 'text' type = 'inline' defaultStyle = 'dark'</h5>
<h4>type: | 'inline' | 'custom' | (defaul = 'inline')</h4>
<h4>render: | string |</h4>
<h4>component: | object |</h4>
<h4>position: | follower | bindToBlock | fixed |</h4>
<h4>customPosition: | object |</h4>
<h4>opacity: | 0 - 1 |</h4>
